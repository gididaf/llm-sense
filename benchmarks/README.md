## Methodology

- All repos analyzed with `llm-sense --skip-empirical --no-cache` (static analysis only)
- Shallow clones (`--depth 1`) to reduce disk/network usage
- Tree-sitter AST analysis enabled for supported languages
- Scores are deterministic for a given repo state + llm-sense version

## Running Benchmarks Locally

```bash
# Run all benchmarks
./scripts/benchmark.sh

# Run a specific repo
./scripts/benchmark.sh express

# Reuse existing clones
./scripts/benchmark.sh --skip-clone
```
